# Restaurant Menu App

A mobile application built with Kotlin for Android that displays a restaurant menu.

## Features

- Browse restaurant menu items
- View item details and descriptions
- Clean and simple mobile UI

## Stack

| Layer | Technology |
|---|---|
| Language | Kotlin |
| Platform | Android |
| Build tool | Gradle |

## Setup

1. Clone the repo
2. Open in Android Studio
3. Run on an emulator or physical device
